## Test cases this task must make pass

### TC-FUNC-007 — Typo-tolerant search ranks flours for atta/aata/attaa
- **Type:** integration
- **Requirement:** FR-5
- **Preconditions:** Meilisearch index seeded with flour SKUs
- **Steps:** 1. Search "atta" 2. "aata" 3. "attaa"
- **Expected:** All three return relevant flours in top 5; per-store availability respected
- **Test data:** `queries: `atta`, `aata`, `attaa``
- **Priority:** P0

### TC-NEG-005 — Nonsense query returns empty state with alternates
- **Type:** integration
- **Requirement:** FR-5
- **Preconditions:** Index seeded
- **Steps:** 1. Search "xqzzt"
- **Expected:** No results; alternate suggestions rendered; no error; < 300 ms
- **Test data:** `query: `xqzzt``
- **Priority:** P1

### TC-PERF-002 — Search p95 < 300 ms including typo queries
- **Type:** load
- **Requirement:** NFR-1, FR-5
- **Preconditions:** Meili seeded as prod-like
- **Steps:** 1. 100 RPS search, 20% typo-variant queries, 30 min
- **Expected:** p95 < 300 ms; relevance spot-checks still pass under load
- **Test data:** `query corpus `perf/search_terms.txt``
- **Priority:** P0

### TC-INT-015 — Stock event propagates to cache + search index ≤ 60 s
- **Type:** integration
- **Requirement:** NFR-7, FR-21
- **Preconditions:** SKU in Redis cache and Meili index
- **Steps:** 1. Zero its stock via inventory API 2. Poll storefront + search
- **Expected:** Cache invalidated and Meili updated ≤ 60 s; SKU stops rendering in browse and search within the window
- **Priority:** P0

### TC-FUNC-025 — Catalog CRUD live within 60 s; bulk CSV upload
- **Type:** integration
- **Requirement:** FR-21
- **Preconditions:** Catalog-role admin
- **Steps:** 1. Edit SKU price + image 2. Upload 200-row CSV 3. Poll storefront
- **Expected:** Storefront (cache + Meili) reflects changes ≤ 60 s; CSV rows applied with success report
- **Test data:** `fixture `catalog/bulk_200.csv``
- **Priority:** P0
